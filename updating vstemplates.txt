how to update vstemplate
-------------------------

STEPS:-

copy a backup of the current project .vstemplate file

load the project

delete the current .vstemplate file

run the build.ps1 with the project loaded

in visual studio select the project in the explorer window

then click on the project icon on the top tool bar and select export template...

go through the process but make sure you untick the import it automatically into visual studio

once the zip file has been created take the .vstemplate file out of it and put it into the project 

get the TemplateData section from your backup of the original and copy over the new one just for that section
